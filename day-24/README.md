# Day 24 - Basic Operators in Shell Scripting

## Objective

What was the goal for today? To understand different types of operators in Bash and how they are used for calculations, comparisons, logic, and file handling.

---

## What I Learned

- Operators are symbols used to perform operations on variables and values.
- Arithmetic operators perform calculations (+, -, *, /, %, ++, --) using $(( )).
- Relational operators compare numeric values (==, !=, <, >, <=, >=) for decision-making.
- Logical operators combine conditions:
- Bitwise operators work on binary values (&, |, ^, ~, <<, >>).
- File test operators check file properties (-e, -d, -r, -w, -x, -s).
