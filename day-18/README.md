# Day 18 -Shell Variable

## Objective

What was the goal for today? To understand what a shell variable does, how to declare it, and methods to use and manipulate it.

---

## What I Learned

- Shell variables store data during script execution
- Types of variables:
    Local variables → exist only in the script
    Environment variables → available to child processes
    Built-in variables → provided by the system (e.g., $HOME, $?)
- Variables are accessed using $variable_name

## What I Built / Practiced

   - Used a read-only variable
   - Attempted to modify the read-only variable (to observe error)
   - Used unset to remove variables
