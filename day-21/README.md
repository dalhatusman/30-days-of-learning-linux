# Day 21 - Loops in Bash

## Objective

What was the goal for today? To understand how to use loops in Bash to automate repetitive tasks, especially for data processing scenarios.

---

## What I Learned

- Loops are essential for automating repetitive tasks like processing multiple files.
- The for loop is used to iterate over a list (numbers, files, sequences).
- The while loop runs continuously as long as a condition is true.

---

## What I Built / Practiced

- I wrote a script to loop through numbers and print outputs.

---

## Challenges Faced

- It is difficult to handle file paths correctly when looping through directories.
---

## Key Takeaways

- Loops are powerful for automation, especially in data engineering workflows
- Control statements (break, continue) give better control over execution.
- Bash loops can replace repetitive manual commands efficiently.
