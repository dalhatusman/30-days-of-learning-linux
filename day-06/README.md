# Day 06 - Archiving and Compressing in Linux

## Objective

What was the goal for today?
To learn how to archive, compress, extract, and manage files in Linux using the tar command.
---

## What I Learned

- tar (Tape Archive) is used to bundle multiple files/directories into one archive.
- It can also compress archives using gzip (-z), bzip2 (-j), or xz (-J).
- It preserves file permissions and directory structure, making it useful for backups.

## Challenges Faced

- Remembering what each flag (-c, -x, -z, etc.) does.
- Compressing files using the zip command required installing zip which needed an administrative/root permission.

---

## Key Takeaways

- tar is essential for backup, packaging, and file transfer.
- Compression is optional but important for saving space.
