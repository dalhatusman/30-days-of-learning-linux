# Day 29 - Role of Linux in Cloud Computing & DevOps

## Objective

What was the goal for today? To understand why Linux is the backbone of cloud computing and DevOps, and how it supports automation and scalability.

---

## What I Learned

- Linux powers most cloud environments due to its stability, scalability, and flexibility.
- Core kernel features like namespaces and cgroups enable containerization and resource control.
- Major cloud providers like Amazon Web Services, Microsoft Azure, and Google Cloud Platform rely heavily on Linux.
- Linux supports virtualization using tools like KVM and Xen.
- Containers (via Docker and Kubernetes) are built on Linux.
