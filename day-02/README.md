# Day 02 - INTRODUCTION TO LINUX COMMANDS

## Objective

What was the goal for today?

To learn and practice linux commands
---

## What I Learned

- How to create and delete files/directories in linux
- Navigating through files/directories
- Commands used for data processing

---

## What I Built / Practiced

- I was able to learn and practice how to create and write into files. I wrote dummy text into a file named story.txt using a simple text editor.
- Copied contents of story.txt into a new file named dummy.txt and deleted the initial file.
- Counted the number of occurrence of a word and saved it into another file using commands like 'grep', 'wc' '>' on a single line.

---

## Challenges Faced

- While I wanted to view into a file, I used the 'cd' command which returned an error and thereafter realized cd command is used exclusively for directories. Then I used the cat command which is used for displaying contents of a file. 


---

## Key Takeaways

- Bash is a powerfull shell which can be used to carry out many commands and interact with the underlying system. It provides the simplicity of using short line of commands to carry out operations that would have taken much more time and effort to execute elsewhere.
