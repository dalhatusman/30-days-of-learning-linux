## What I Learned

- Linux operating system is an open-source UNIX-like OS. Open source meaning, anyone is free to use, customize and redistribute which has led to several distributions.
- Most Common examples include: Ubuntu: a user/beginner-friendly linux distribution ideal for server ,personal use, cloud-computing,etc. Debian: a reliable distribution used for servers.  Kali: A security-focused distribution mostly used in the cybersecurity domain for ethical hacking, penetration testing, etc.
- How Linux is used in different domains; CyberSecurity, Development, Server, Cloud Computing, IoT systems, Supercomputers, etc.
