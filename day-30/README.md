# Day 30 - Bash Task 

## Objective

What was the goal for today? To automate the setup of a Python project environment using a Bash script to eliminate repetitive manual configuration for new developers.

---

## What I Learned

- How to automate environment setup using Bash scripting.
- The importance of virtual environments (.venv) in Python projects.
- How to structure scripts using functions and a main entry point.
- Writing idempotent scripts (safe to run multiple times).
- Creating and managing .gitignore files programmatically.

---

## What I Built / Practiced

- Built a setup.sh script that:
    - Creates or activates a .venv
    - Upgrades pip
    - Generates a .gitignore
    - Installs packages like pandas and requests
    - Logs all actions to setup.log
- Implemented colored terminal outputs for better user experience.

---

## Challenges Faced

- Handling cross-platform issues (Linux vs Windows virtual environment paths).
- Managing logging while still displaying output in the terminal.

---

## Key Takeaways

- Automation saves time and reduces human error in development workflows.
- Bash scripting is powerful for DevOps-style tasks.
