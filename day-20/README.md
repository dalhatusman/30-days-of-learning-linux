# Day 20 - Bash Variables and Their Types

## Objective

What was the goal for today? Understand how variables work in Bash, including how to define, use, and manage different types of variables in scripts.

---

## What I Learned

- Bash variables are used to store and manage data during script execution. They are untyped and can hold text, numbers, command output, or arrays. There are different types such as user-defined, environment, and special variables. Variables can be global (accessible everywhere) or local (restricted to functions)


---

## What I Built / Practiced

- Created a script with declared variables and read-only variables
- Practiced unsetting variables and attempting to modify read-only variables


---

## Challenges Faced

- Keeping track of correct syntax 

---

## Key Takeaways

- Variables are essential for making scripts dynamic and reusable
- Always avoid spaces in variable assignment
- Read-only variables cannot be modified or unset
- Proper understanding of scope prevents unexpected behavior
