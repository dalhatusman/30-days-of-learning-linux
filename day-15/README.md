# Day 15 - Broadcasting Messages in Linux

## Objective

What was the goal for today? Learn how to send messages to users in a Linux system using shell scripts.

---

## What I Learned

- Linux allows system-wide communication using commands
- wall is used to broadcast messages to all logged-in users.
- write is used to send a message to a specific user.
- Broadcasting is useful for system alerts, maintenance notifications, and admin communication

---

## What I Built / Practiced

- Wrote a simple shell script to broadcast a message using wall.


---

## Challenges Faced

- Could not fully test broadcasting features due to permission restrictions and limited system access.

---

## Key Takeaways

- Linux provides built-in tools for real-time communication between users and these tools are important for system administration and monitoring.
