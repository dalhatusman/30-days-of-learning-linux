# Day 03 - Linux Commands

## Objective

What was the goal for today?
To Practice more linux commands 
---

## What I Learned

- Redirection and Pipes in Linux: Chaining commands and redirecting the output of one file to another.
- Environment variables: used to store user-specific and system settings
- More commands used to perform various tasks in Linux

---

## What I Built / Practiced

- Practiced linux commands like whoami to reveal the current user on the terminal 
- Linux commands used to redirect/append output to a file and read input from a file.
- Combining and running multiple commands and the  logical order in which they are executed.

---

## Challenges Faced

- Tried creating a file with the touch command in uppercase which returned a command not found error due to case sensitivity

---

## Key Takeaways

- Linux is a case-sensitive Operating System
- In a situation where one is unsure what a command does, using the 'whatis' command is a quick way to know what it does.
